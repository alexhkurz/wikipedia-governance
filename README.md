# Wikipedia Governance

This repo is intended to document research on Wikipedia Governance at Chapman University. It originated from the [1 credit](https://en.wikipedia.org/wiki/Course_credit#United_States) course CPSC 298 [Wikipedia Governance](https://en.wikipedia.org/wiki/Wikipedia:Administration) in Fall 2025 at Chapman University, see [CPSC 298 Wikipedia Governance]() for the course repo.
